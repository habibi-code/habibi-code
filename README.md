### 📌 Flagship Project

#### 🚀 ChatSooq — AI-Powered Conversational Commerce
An AI-driven commerce ecosystem built to help small business owners in Africa automate messaging-based sales (Telegram/WhatsApp) without forcing them to deal with the overhead of traditional websites.

*   **The Problem:** Small merchants suffer massive drop-offs due to slow manual response times, chaotic order flows inside chat windows, and repetitive FAQ handling.
*   **The Solution:** Generates ultra-lightweight, high-performance digital stores instantly, backed by an **AI Sales Assistant** built to handle context-aware customer queries, navigate catalog searches, and streamline orders 24/7.
*   **Architecture Stack:** Next.js / React interactive storefronts seamlessly integrated into robust Express & Node.js backends, powered by Prisma ORM and PostgreSQL data models, alongside targeted Python AI processing modules.

👉 **[Explore the ChatSooq Codebase](https://github.com/habibi-code/chatsooq)**
